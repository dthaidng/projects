Installation:

	Just extract, and run ;)

Usage:
	Just right-click on the symbol and select either to enable or
	disable the windows keys.
	
	If the tray icon is gray, the windows keys are turned off,
	if it is colored then the windows keys are enabled.

	To switch between On and Off state more quickly, you can just
	double click the icon.

	For even quicker switch, you can use the Shortcut: CTRL+ALT+END

Uninstall:
	Quit program, and delete it ;)
